Laza is a fully functional e-commerce mobile application built with **Flutter** and **Firebase**. It features user authentication, product browsing, a favorites system, and a shopping cart with mock checkout functionality. The project demonstrates secure mobile development practices and includes automated UI testing using **Appium**.

### Key Features
* **Authentication:** Secure Signup & Login (Email/Password) via Firebase Auth.
* **Product Catalog:** Browse products with details retrieved from Firestore.
* **Cart System:** Add items to cart, view total price, and simulate checkout.
* **Favorites:** Users can save and remove items from their wishlist.
* **Automated Testing:** Python-based Appium scripts for end-to-end testing.
